# 🚓 SecureCheck - Traffic Stops Analysis & Prediction

## 📌 Overview
This project analyzes traffic stop data and predicts violation outcomes using historical data.

## ⚙️ Tech Used
- Python
- Pandas
- Streamlit
- SQL (TiDB)

## 📊 Features
- Data analysis dashboard
- SQL-based insights
- Prediction system
- Natural language output

## 🚀 Run the app
streamlit run app.py
